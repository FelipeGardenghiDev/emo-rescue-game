Emo Rescue

Guie a garota ruiva por 3 fases curtinhas para salvar o garoto emo.

  • Setas ou A / D para andar
  • W, Seta para cima ou Espaço para pular
  • Enter para começar ou reiniciar
  • P ou Esc para pausar
Para jogar melhor no celular, vire na horizontal ou use tela cheia.
