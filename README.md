# Emo Rescue

> Mini gamezinhu `<3` feito em `HTML`, `CSS` e `JavaScript`, pronto para rodar no `GitHub Pages`.

Em **Emo Rescue**, você guia a garota ruiva por **3 fases curtinhas** para salvar o garoto emo em uma noite cheia de neon, portais, clima dramático e trilha procedural.

## Preview

- visual neon com vibe emo
- gameplay de plataforma 2D
- feito para funcionar bem em desktop e mobile

## Features

- **3 fases** com progressão própria
- **menu inicial** com seleção de fase
- **checkpoints** e respawn ajustado
- **trilha procedural** com variação por fase
- **efeitos sonoros** com botão de mute
- **pause**, **cutscene final** e **créditos**
- **parallax**, clima, partículas e efeitos visuais
- **bloom fake**, **vinheta**, **scanlines** e glow no portal
- **touch controls** para celular
- **tela cheia**

## Controles

### Desktop

- `←` / `A` — andar para a esquerda
- `→` / `D` — andar para a direita
- `↑` / `W` / `Espaço` — pular
- `Enter` — começar ou reiniciar
- `P` / `Esc` — pausar

### Mobile

Use os botões de toque na tela. Para uma experiência melhor, jogue em **modo horizontal** ou ative a **tela cheia**.

## Como rodar

Como o projeto é estático, basta abrir o arquivo `index.html` no navegador.

Se preferir, você também pode servir a pasta com qualquer servidor estático simples.
